# codespeak
Personal programming language

## Expression

Examples:

`()` means symbolic computation (or sub-expression).
Whitespace needed to separate words.

`2`

`(2 * 3)`

`2 * 3` implicit top-level `()`

`2 * (3 * 2)`

`,` means list

`2, 3` list of 2 natural numbers

`2, (3 * 4), 4`

`2, (3,4),4`
